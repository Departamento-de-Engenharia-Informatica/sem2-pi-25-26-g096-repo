# US02 - Accept/Reject Registration Requests

## 1. Requirements Engineering

### 1.1. User Story Description

As an Administrator, I want to accept/reject registration requests on the platform.

### 1.2. Customer Specifications and Clarifications

**From the specifications document:**

> The system must allow an Administrator to accept or reject registration requests.

**From the client clarifications:**

> **Question:** Who can accept or reject registration requests?
>
> **Answer:** Only Administrators.
